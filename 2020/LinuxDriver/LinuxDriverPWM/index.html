PWM 概述

  • Ton：信号高电平持续时间
  • Toff：信号底电平持续时间
  • Period： 完整 pwm 周期
  • Duty cycle：pwm 信号周期内保持为 ON 的时间百分比

Linux 下 PWM 驱动

数据结构

pwm 的数据结构如下图所示：原图

  • pwm_chip 结构体用于抽象 soc 的 pwm 控制器
  • pwm_device 用来抽象设备
  • pwm_state 表示 pwm 设备的周期、占空比、极性等信息
  • pwm ops 结构体提供一系列回调函数。这些回调函数的操作对象是具体的 pwm device
